# Personal Walk ETA Lab

평균 보행 속도(약 4~5km/h)에만 의존하는 지도 ETA를 개인 보행 특성으로 다시 계산하는 정적 웹 프로토타입입니다.

## 핵심 기능

- 사용자 프로필(키, 체중, 보폭) + 스마트폰 센서 입력(걸음 수, GPS 속도)을 동시에 수집
- `보폭 × 분당 걸음 수`를 기반으로 개인 보행 속도를 계산
- 경사도, 시간대, 피로도, 최근 ETA 오차율을 반영한 AI 보정 계수 적용
- 평균 지도 ETA와 개인화 ETA를 한 화면에서 비교
- 최근 7일 학습 추세 차트로 속도 변화를 시각화

## 실행 방법

이번 버전부터는 저장소 안에 **로컬 정적 서버(`server.js`)** 를 포함했습니다.
브라우저에서 `localhost` 접속이 필요하면 아래 명령으로 서버를 먼저 실행하세요.

```bash
npm start
```

기본 접속 주소는 아래입니다.

```text
http://127.0.0.1:3000
```

포트를 8000으로 바꾸고 싶다면 아래 명령을 사용하면 됩니다.

```bash
npm run start:8000
```

## 파일만 바로 열기

서버를 띄우지 않고도 `index.html` 파일을 브라우저에서 직접 열어 화면을 확인할 수 있습니다.
다만, 브라우저 미리보기/포트 포워딩 환경에서는 보통 `npm start` 방식이 더 안정적입니다.
